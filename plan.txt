ML Plan: The UCI ML database has a dataset that essentially lets you use ML to generate a confidence score for credit worthiness of a specific applicant. The issue with this is that it is credit card independent (meaning the approval / denial target feature does not consider which credit card that applicant is applying for). To circumvent this, we can create a second model that predicts the minimum credit score required for a specific card (this data can be scraped off nerdwallet) based off the features of the card. This we basically allow us to determine how difficult a given card is to get accepted to, and we can then line those credit score minimums up with credit worthiness scores to approximate credit card approvals for a given card.
